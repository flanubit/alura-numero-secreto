## Alura - Jogo do número secreto ##

Um projeto do curso de lógica de programação da __Alura__, no curso de nivelamento DevOps.

- O arquivo __.gitignore__ informa que 'log.txt' deve ser ignorado.
- O arquivo log.txt contém a saída do código abaixo:
~~~~
git log > log.txt
~~~~
